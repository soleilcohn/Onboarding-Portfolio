Onboarding Portfolio – WRT 307 Service Learning   
Last Edit: 2026-02-12 
============================================================

This project was created by Soleil Cohn as an individual Onboarding Project for WRT 307: Writing for the Professions at Syracuse University.

This project involved developing a professional onboarding portfolio that introduces me as a candidate in a workplace setting. The portfolio includes a collection of written and rhetorical materials designed to demonstrate communication skills, audience awareness, and professional presentation.

This repository contains the portfolio website developed to present all onboarding materials in a clear, accessible digital format.

Live Site:  
https://soleilcohn.github.io/Onboarding-Portfolio/

----------------------
PROJECT PURPOSE:
----------------------

The objective of this assignment was to create a professional onboarding portfolio that simulates the experience of applying for and entering a workplace environment. This included producing multiple workplace-style documents and presenting them in a cohesive, well-designed digital portfolio.

The project emphasizes:
- Professional self-presentation and communication  
- Rhetorical awareness and audience analysis  
- Writing across multiple workplace formats  
- Clear organization and digital portfolio design  
- Translating written content into a structured online format  

----------------------
WORK COMPLETED:
----------------------

As part of this individual project, I:

- Wrote a professional introductory bio tailored to a specific audience  
- Conducted a rhetorical analysis of an organization and its communication style  
- Developed an application letter demonstrating audience awareness  
- Created an unsolicited recommendation letter addressing a real-world issue  
- Revised and analyzed a workplace memo based on effectiveness and tone  
- Designed and built a portfolio website to present all materials cohesively
