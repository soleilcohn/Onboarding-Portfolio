Instructions Portfolio – WRT 307 Service Learning Project
Last Edit: 2026-02-12  
============================================================

This project was created by Soleil Cohn for WRT 307: Writing for the Professions at Syracuse University.

This project is part of the Instructions Project, a course assignment focused on developing clear, user-centered instructional materials. The project required the creation of a digital portfolio that presents step-by-step instructions supported by visuals and usability testing.

This repository contains the portfolio website developed to present the instructional content and project components.

Live Site:  
https://soleilcohn.github.io/Instructions-Portfolio/

----------------------
PROJECT PURPOSE:
----------------------

The objective of this assignment was to design and present a clear, visually supported set of instructions for a user to complete a task. The project focused on combining written directions with effective visuals to improve usability and understanding.

The project emphasizes:
- Writing concise, step-by-step instructions  
- Using visuals as functional communication tools  
- Designing a clear and navigable digital interface  
- Applying user-centered design principles  
- Revising content based on usability and clarity  

These goals reflect the broader purpose of WRT 307, which focuses on professional communication, document design, and audience-centered writing :contentReference[oaicite:0]{index=0}.

----------------------
WORK COMPLETED:
----------------------

As part of this individual project, I:

- Designed a structured set of step-by-step instructions  
- Created visual aids, including labeled diagrams and images, to support each step  
- Organized content into a logical and easy-to-follow sequence  
- Developed a fully functional website to present the instructions  
- Integrated design elements to improve readability and user experience  
- Applied usability principles to ensure clarity and effectiveness
